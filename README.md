# Basedbook
## Description
[BasedBook](https://basedbook.herokuapp.com) is a social network login template that follows the visual design of [Facebook](https://www.facebook.com) and other social networks.

## Dependencies 
* npm 
* babel
* webpack
* React
* express
* bcrypt
* mongoose
* nodemon
